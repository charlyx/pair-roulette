## Une idée, une histoire

**Pair-Roulette**, c'est avant tout une idée.  
Une idée que j'ai eu lorsque j'ai proposé à tous les collègues du groupe pour 
lequel je travaille.


> Hi all 👋
> 
> I’ve just had an idea and I’d like to share it with you.
> I would like to pair program with you. You get to choose the project and the language as long as it is open source.
> 
> I’m sure we’ll all learn something!
> What do you say? 😃
> 
> I look forward to hearing from you.

De l'enthousiasme suscité par cette proposition est née l'idée de **Pair-Roulette**. 

**Pair-Roulette** est donc une plateforme de mise en relation pour les personnes
souhaitant contribuer à des projets Open-Source en collaboration.


Son fonctionnement est le suivant :

 * 🔐 Connectez-vous grâce à votre compte GitHub
 * 🖋 Renseignez vos langages préférés et les langages que vous souhaitez 
 découvrir
 * ☸️  Lancez la roulette
 * 📫 Votre pair a reçu une invitation
 * 🤝 C'est parti, l'invitation a été acceptée : vous prenez connaissance de votre 
 pair
 * 💬 Entrez en contact sur [gitter](https://gitter.im/pair-roulette/community) 
 grâce à votre pseudo GitHub
 * 🗂 Choisissez ensemble le projet et l'issue sur laquelle vous allez contribuer
 * 👩‍💻👨‍💻Créez votre PR : n'oubliez pas d'ajouter votre pair avec 
 [Co-authored-by](https://help.github.com/en/articles/creating-a-commit-with-multiple-authors)
 * 💌Envoyez-nous le lien de votre PR
 * 👍👎 Faites une revue de votre collaboration
 * 🥳 Lancez à nouveau la roulette et amusez-vous !
